Ley Clara

Asistente legal inteligente para Uruguay que traduce leyes a lenguaje simple y ayuda a entender derechos, obligaciones y trámites de forma rápida.

---

🚀 Demo
https://leyclara.vercel.app

---

📌 Features

- 🧠 Respuestas en lenguaje claro
- ⚖️ Basado en leyes de Uruguay
- 💬 Chat interactivo estilo asistente
- 📚 Explicación + ejemplos prácticos
- 📱 Diseño responsive

---

🛠️ Tech Stack

- React
- Tailwind CSS
- Framer Motion
- Lucide Icons

---

🎯 Objetivo

Facilitar el acceso a la información legal para cualquier persona sin conocimientos jurídicos.
